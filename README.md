# Art Collections

A private study archive of two curated art collections, published as a static site.

- **Weimar Republic Art** (`gallery.html`) — German art 1918–1933.
- **Abstract Modern Art** (`modern_gallery.html`) — non-objective painting & sculpture.

Landing page: `index.html`. Images live in `weimar/` and `modern_art/`, with
`MANIFEST.json` in each folder recording artist / title / year / source for every work.

Sourced from Wikimedia Commons, Wikidata, WikiArt and Europeana for private, educational use.
